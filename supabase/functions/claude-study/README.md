# claude-study

Supabase Edge Function that accepts parsed knowledge sources and returns Study Canvas JSON DSL generated by Claude.

## Secrets

Required:

- `ANTHROPIC_API_KEY`

Optional:

- `ANTHROPIC_MODEL` (default: `claude-3-7-sonnet-latest`)

## Deploy

```bash
supabase functions deploy claude-study
supabase secrets set ANTHROPIC_API_KEY=your_key
# optional
supabase secrets set ANTHROPIC_MODEL=claude-3-7-sonnet-latest
```
